## 🎉 博客上线！

历经环境配置与主题调优，个人站点**风声鹤唳的Island**今日上线，以技术笔记与日常分享为主要内容。

### 🛠️ 技术栈

| 组件 | 选型 |
|------|------|
| 框架 | Hexo 8.x |
| 主题 | Butterfly |
| 托管 | GitHub Pages |
| 部署 | GitHub Actions |
| 评论 | Giscus |
| 搜索 | Local Search |
| 统计 | Google Analytics |

### 📝 搭建过程

1. **Hexo 初始化** —— 安装 Node.js，使用 `hexo init` 初始化项目
2. **主题配置** —— 选择 Butterfly 主题，配置导航栏、代码高亮、暗色模式
3. **GitHub Pages 部署** —— 配置 GitHub Actions 实现推送自动部署
4. **功能插件** —— 评论系统、本地搜索、统计分析
